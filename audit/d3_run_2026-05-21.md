**Findings (2026-05-21):**

- [x] **Equity curve** — Monthly equity table embedded above. IS draws
      down to −1,041 bps over 9 months (Aug 2023 → May 2024); OOS draws
      down −790 bps in 5 weeks (Mar–Apr 2026). No single dominant
      trade/week — drawdowns accumulate across many losing trades, as
      expected from a 96.6%-trail-exit strategy. Pattern is consistent
      with "small edge slowly eaten by trail-induced friction."
- [x] **PnL distribution** — Skew +1.8 to +2.1, kurt excess 7.6–10.8.
      Fat-tailed positive — a small number of trades produce
      disproportionately large positive PnL (the SESSION_CLOSE bucket).
      Sharpe under-quantifies the tail; DSR's denominator partially
      adjusts for it (skew/kurt-aware) and still rejects decisively.
- [x] **≥20 random trade sample** — 20 charts per epic in
      `d3_trade_inspection.ipynb` (`audit/build_d3_inspection_nb.py`
      regenerates). Chandelier trail trajectories overlay each chart as
      a stepped line. Visual review confirms the trail follows the
      running extreme as designed, with occasional "loosen" steps when
      ATR expands. The trail mechanism is operating per the spec — no
      implementation defect.
- [x] **Drawdown periods** — As above, no single trade or week
      explaining the result. Drawdowns are population-level.
- [x] **Parity test re-run** — `pytest tests/unit/test_parity.py`
      passes (5/5). Engine ≡ live monitor per-bar AND engine ≡
      closed-form `chandelier_stop()` formula. No drift.
- [x] **Sanity vs literature** — Realized per-trade SR ≈ 0 (US500) /
      +0.07 (DE40) on M15 retail CFD, cost-included. Zarattini SPY at
      1-min reports SR 1.33 *un-re-costed* on futures/ETF. Our result
      is "substantially lower," exactly the BR-caveat the catalog
      flagged. Outright unprofitable after retail-CFD friction is
      consistent with the D1 cost-anchor prediction.
